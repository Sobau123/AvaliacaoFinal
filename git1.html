Git Bash

O que é ?

- É a ferramenta de versionamento local, que salva o estado do arquivo na hora do versionamento e se o arquivo não houver alterações ele cria um link simbólico para o arquivo não editado.

Como usa e configura ?

No Git bash:

- Para colocar seu nome e e-mail você usará:

$ git config --global user.name "Seu Nome" (Para colocar seu nome)
$ git config --global user.email "seuEmail@gmail.com" (Para colocar seu e-mail)

Para você entrar em uma pasta ou criar uma pasta você usará:

$ cd Documents (Para entrar na pasta Documents)
$ mkdir "Nome do arquivo" (Para criar a pasta)

Para você ver o que tem dentro da pasta basta usar:

$ ls "Nome da Pasta"

Para ter a permissão para mexer em um arquivo você terá que usar:

$ git init

E para entrar direto no Visual Studio Code usará:

$ code .
