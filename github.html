GitHub

O que é ?

É um serviço que usa o Git Bash para versionamento, ou seja, é um local de armazenamento de códigos / repositório remoto.

Para que é usado ?

Pode usado para impresas que mexam em sites possuindo um tipo de separação de trabalhos onde possue vários tipos de setores como:

Main (É a principal ou seja a parte onde vai ser liberado para os clientes / público-alvo)
Release/Hml (É o termino do site onde é revisado para ser liberado para a main)
Dev (É o desenvolvimento ou seja onde terá os ajustes e criação do site)
BugFix (É onde arruma os erros no sistema ou no site)
Feature (É onde a maioria das empresas menos usa)

Como configura a chave SSH:

Para criar sua chave SSH teremos a seguinte ordem:

Verificar se existe chave
$ ssh.ls -al ~/.ssh

Criar uma nova chave. (ID)
$ ssh-keygen -t ed25519 -C "Seu e-mail usado pra configura o Git Bash"

Inicializar agente-ssh.
$ eval "$(ssh-agent -s)"

Adicionar chave ssh ao agente.
$ ssh-add ~/.ssh/id_ed25519

Copiar chave ssh.
$ clip < ~/.ssh/id_ed25519.pub Adicionar chave no github Vai no Github, procure Settings, depois SSH and GPG keys, agora New SSH key, coloque um titulo adequado ao branch e aperte Ctrl+V no ssh e crie ossh.

Adicionar chave no github
Vai no Github, procure Settings, depois SSH and GPG keys, agora New SSH key, coloque um titulo adequado ao branch e aperte Ctrl+V no ssh e crie o ssh.

Testar conexão
$ ssh -T git@github.com em seguida perguntara se você quer autoriza colocar a chave no Github se você quiser digite "yes" se não digite "no" .

Para criar um repositório e conseguir mexer no arquivo terá que:

Primeiro - entrar no GitHub e no lado de "Top Repositories" terá um botão escrito "New" onde você entrará na tela de novo repositório

Segundo - colocar um nome pro seu repositório e uma descrição e marcar o public e o add README file e depois clicar em "create repository"

Terceiro - clicar em "<> code" e ir em SSH aonde terá embaixo "git@github.com..." onde você copiará isso

Quarto - Você vai ir no Git Bash e usar o código "$ cd Documents" (Para entrar na sua pasta Documents) e depois usar "$ git clone para ter seu repositório na pasta Documents" se você não lembra o nome do repositório pode usar o comando "$ ls (Dentro da pasta Documents para ver o nome da pasta do repositório)" e então usar "$ cd "Nome do repositório" para entrar na pasta do repositório e então usar "$ code ." para entrar no Visual Studio Code.
