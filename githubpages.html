GitHub Pages

Para que serve ?

Fornece suporte para sites estáticos, básicos um exemplo um home page

Como colocar seu repositório no GitHub Pages

Primeiro - Acesse seu repositório estático e clique em settings(configurações)

Segundo - Clique em pages selecione a branch para publicar e salve em seguida clique em "visit site"(visitar o site).


Como criar um exemplo de React

Onde será referente ao estilo react e para configura será feito:

Primeiro - Crie um novo repositório com o nome "teste-react" público e com README e depois execute as funções de git clone (que tem uma explicação na página de "GitHub")

Segundo - Dentro do terminal do VScode digite "npx create-react-app" depois adicione na raiz do projeto um index.html(com o básico do HTML) e então depois suba todos os arquivos e commit.(Para ver se já está funcionando seu react basta colocar no terminal do VScode "npm start" e se aparacer um símbolo de química quer dizer que está funcionando)

Terceiro - Para colocar no GitHub Pages você não poderá colocar a branch da main se não aparecerá seu README então você terá que ir na raiz denovo e criar uma pasta ".github" dentro dela uma pasta "workflows" e dentro dessa um arquivo "build.yml" para ver o que acontecerá nas actions basta commitar e ver nas actions no GitHub e ver se está dando build e deploy

Quarto - um possível erro que pode acontecer é o 403-sem permissão mas para corrigir é simples basta ir no GitHub depois em settings no seu repositório de teste-react clicar em "Actions" depois "General" e então procurar por "Workflow permissions" e marcar "Read and write permissions" e "Allow GitHub Actions to create and approve pull requests" e estará resolvido.

Quinto - Mas não acabou ainda no GitHub Pages aparacerá outra branch no qual será selecionada "gh-pages" mas não funcionará pois não está ainda configurado na pasta certa, então teremos que ir no VScode denovo e ir em "package.json" e embaixo de version criar "homepage(entre aspas): "teste-react/"" e então basta commitar denovo pro GitHub e estará tudo pronto
